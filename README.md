# sthira-marketing

Marketing site for **Sthira by SOS Markets** — a SEBI-conscious retail
investment platform for layman investors. Sthira does goal-based SIPs and
pre-authorized drift rebalancing. It never lets the LLM trade for you.

Astro 6 + Tailwind v4 + MDX. Static output served by nginx on Coolify.

## Local dev

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # static output to dist/
```

## Deployment

Coolify pulls from `origin/main`. Hosted at https://sthira.sosservices.online.

## Brand

- Product name: **Sthira** (Sanskrit: steady, stable)
- Eyebrow: A Product by SOS Markets
- Palette: emerald (`emerald-700` as primary CTA)
- Tone: plain English for a layman retail investor — no operator-grade jargon
